# ⚽ Prospect FC

A full-stack football website built to represent **Prospect FC** — combining clean frontend design with real backend functionality and database integration.

---

## Overview

Prospect FC is a dynamic web application designed to showcase a football team, its players, fixtures, and overall brand.

Built as a full-stack project, it demonstrates the ability to connect a responsive user interface with a working backend and structured database.

---

##  Features

*  Modern, responsive homepage
*  Team & player profiles
*  Fixtures and match information
*  Merchandise section 
*  Team stats
*  Newsletter
*  Article and blogs about matchdays
*  Dynamic content powered by backend + database

---

##  Tech Stack

### Frontend

* HTML5
* CSS3
* JavaScript

### Backend

* Node.js
* Express.js

### Database

* SQL (MySQL / SQL Server)

---

##  Setup & Installation

1. Clone the repository:

```bash
https://github.com/mrgovernment-max/ProspectFC.git
```

2. Navigate into the project:

```bash
cd Prospect-fc
```

3. Install dependencies:

```bash
npm install
```

4. Open in your browser:

```
http://localhost:3000
```

---

## 📂 Project Structure
```
/public        → Frontend (HTML, CSS, JS)

```
